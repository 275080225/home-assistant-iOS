Se acabó el verano, vuelta a la escuela, lo que significa que ya es hora de que se actualice la app de HomeAssistant para iOS. Volvemos a la normalidad y estamos trabajando en la implementación de nuevas funciones y en la eliminación de errores una vez más. Disculpas por el parón durante el verano. 

 Estas son las novedades de esta versión: 

 - Compatibilidad con iPhone X (en mi humilde opinión HA se ve muy bien) 
 - Tipo de contenido adjunto de categoría de cámara predeterminado a JPEG 
 - El color de la barra de estado cambia con el color del tema de la interfaz 
 - Soporte de orientación horizontal 
 - Diálogos de permiso mejorados 
 - Mejora la comunicación API con Home Assistant 
 - Cambiado el enlace de soporte de chat de Gitter a Discord 
 - Añadido el numero de versión de la app a la página "Acerca de" 
 - Nuestra primera traducción: alemán! 
 - Mejora la gestión de la ubicación 
 - Swift 4 
 - Eliminado un montón de código obsoleto y dependencias 
 - Se eliminó la interfaz de usuario nativa. A partir de ahora,se usará la vista web, ya que Home Assistant agrega nuevos componentes / plataformas con demasiada frecuencia para que la interfaz de usuario nativa se mantenga al día. 
 - Un montón de mejoras en la vista web en torno al tamaño / diseño y la lógica 
 - Mejoras en la barra de estado para problemas de paisaje y tamaño 

 No olvides dejar una revisión actualizada ya que iOS 11 restablece las revisiones con cada nueva versión. ¡Gracias!
