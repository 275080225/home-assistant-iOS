挥别了慵懒的盛夏，我们循着秋的步伐迎来了新的学期（啦啦啦~~~！）
假日归来，各位看官们久等了。在新的学期里，我们定会一洗假日的慵懒，以百倍的热情投入到 Home Assistant 项目当中，努力推陈出新。

此次版本更新包括：

- 适配 iPhone X（恕我直言，HA在这上面用起来超顺滑）
- 将默认相机类别的附件内容格式设置为 JPEG
- 状态栏颜色随着前端主题的颜色而变化
- 支持横屏显示
- 改善权限对话框
- 改善与 Home Assistant 的 API 通信
- 聊天链接已由 Gitter 改为 Discord
- 将版本号添加到“关于”页面
- 推出第一个外语版本：德语！
- 改善位置管理功能
- 支持 Swift 4
- 删除部分陈旧代码
- 删除本地 UI。用户过于频繁地添加新的组件和平台已然成为常态，因此从现在起它会变为网页视图。
- 围绕着 Web 视图的大小、布局和逻辑的大量的改进
- 改善状态栏的横屏显示以及缩放问题

每一版更新，iOS 11都会重置评论，所以不要忘了在每一个新版本下留下您的评论。谢谢!
